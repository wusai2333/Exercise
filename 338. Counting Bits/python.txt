class Solution(object):
    def countBits(self, num):
        """
        :type num: int
        :rtype: List[int]
        """
        res = [0]*(num+1)
        for i in range(1,num+1):
            res[i] = res[i>>1] + (i & 1)
        return res

An easy recurrence for this problem is f[i] = f[i / 2] + i % 2
