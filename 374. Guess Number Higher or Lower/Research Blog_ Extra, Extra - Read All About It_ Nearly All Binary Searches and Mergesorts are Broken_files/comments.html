+Alina Barnett Agreed, but it's rather easy to fix. Actually fixing it removes code. What about this one ? Did I still overlooked some case ?

public static int binarySearch(int[] a, int key) {
     int low = 0;
     int high = a.length;
     while low < high {
         int mid = low/2 + high/2;
         int midVal = a[mid];
         if (midVal < key)
            low = mid + 1
         else if (midVal > key)
            high = mid;
         else
            return mid; // key found
     }
     return -(low + 1);  // key not found.
}﻿
